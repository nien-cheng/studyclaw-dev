# 实战案例：搭一套我的 AI 个人助理系统

学了很多概念，装了很多技能——但怎么把它们串起来，真正变成每天在用的工具？

这一篇用一个具体案例，串联所有技能，搭一套**真正可用的个人 AI 助理**。

## 我的需求

- 早上 8 点收到**今日简报**（新闻 + 天气 + 日程）
- 在**飞书**上随时发消息给它
- 它记住我的**工作背景和偏好**
- 需要搜索时能**联网查资料**
- 重要邮件来了**及时通知我**

## 系统架构

```
┌─────────────────────────────────────────────┐
│              飞书（主入口）                   │
└─────────────────┬───────────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────────┐
│         OpenClaw Gateway                     │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐       │
│  │ 主Agent │ │ 资讯Agent│ │ 邮件Agent│       │
│  └────┬────┘ └────┬────┘ └────┬────┘       │
│       │            │           │            │
│  Memory      Skills         CronJob         │
└───────┼────────────┼───────────┼────────────┘
        │            │           │
        ▼            ▼           ▼
   分层记忆     Tavily搜索    定时任务
```

## 第一步：基础配置（已完成）

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
```

配置好飞书 Channel（参考《接起飞书/Telegram》），确保消息能收发。

## 第二步：设定人格

创建 `~/.openclaw/workspace/SOUL.md`：

```markdown
# 我是谁
- 名字：Moon
- 风格：直接、靠谱、不废话
- 优先给结论，再补充解释

# 工作边界
- 重要决策先问我，不自己做主
- 不知道的事说不知道，不瞎编
- 敏感操作（删除、发送外部）必须确认
```

创建 `USER.md`：

```markdown
# 关于我的主人
- 名字：郑念
- 做 OpenClaw 中文教程
- 需要每天了解 AI / 科技动态
- 希望早上 8 点收到简报
```

## 第三步：安装必要技能

```bash
# 搜索
openclaw skills install tavily-search

# 天气
openclaw skills install weather

# 飞书日历（需要飞书开放权限）
openclaw skills install feishu-calendar

# 邮件
openclaw skills install email-summary
```

## 第四步：配置定时任务

在 `openclaw.json` 的 `crons` 部分添加：

```json
{
  "crons": [
    {
      "id": "morning-brief",
      "schedule": "0 8 * * *",
      "task": "给我发今日简报，包含：\n1. 北京/上海天气\n2. 今日科技/AI 新闻（3条）\n3. 今日待办日程\n格式简洁，用 Emoji 分隔"
    }
  ]
}
```

## 第五步：测试

手动触发一次简报，验证流程：

```
用户：发一下今日简报

AI：
🌤 北京：晴，18-26°C
📰 科技新闻：
   1. OpenClaw 发布 v2.3...
   2. Anthropic Claude 4...
   3. ...
📅 今日待办：
   - [ ] 审校教程 T01-T04
   - [ ] 更新博客
```

## 效果

有了这套系统之后：

- **每天早上 8 点**：自动收到简报，不用打开任何 App
- **任何时候**：飞书发消息，立刻响应
- **搜索任何问题**：它联网查资料，给出答案
- **用得越久**：它越懂你的偏好和习惯

**这就是 OpenClaw 的目标——成为你真正离不开的数字助理。**

## 怎么扩展？

当前系统还可以继续加：

- 📊 股票监控（跌了涨了自动通知）
- 🖼 生图功能（给文章配图）
- 📝 自动写文章草稿
- 🔔 GitHub Issue 管理

**这套系统的本质是：你用什么工具，OpenClaw 就能接入什么工具。你有什么需求，它就能学会什么技能。**
