OpenClaw（小龙虾）是一个开源的个人 AI 助手框架，让你把 Claude、GPT 等大模型接入微信、飞书、Telegram，变成真正能帮你干活的私人助理。

## 你需要准备什么

- 一台服务器（或本地电脑）
- Node.js 18+
- 一个大模型 API Key（Claude / OpenAI / MiniMax 都行）

## 安装

```bash
npm install -g openclaw
openclaw setup
```

按向导填入你的 API Key，选择连接的渠道（飞书/Telegram/微信），几分钟就跑起来了。

## 第一次对话

安装完成后，在飞书或 Telegram 找到你的 bot，发一条消息：

> 你好，你是谁？

AI 助手会自我介绍，并开始记住你的偏好。
