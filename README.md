# StudyClaw.dev

OpenClaw & AI Agent 中文教程站

## 关于

StudyClaw.dev 是专注于 OpenClaw、Hermes Agent 等 AI Agent 工具的中文教程站，帮助中文用户快速上手、深度使用 AI 助手。

## 内容方向

- OpenClaw 入门到进阶
- Hermes Agent 使用指南
- AI Agent 变现实战
- 工具对比与选型

## 本地开发

```bash
npm install
npm run dev
```

## 部署

通过 Cloudflare Pages 自动部署，push 到 main 分支即可。
